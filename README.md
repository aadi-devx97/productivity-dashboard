# Productivity Dashboard

A modern productivity dashboard built with React as part of my Full Stack + AI learning journey.

---

# Features

- Add, delete and complete tasks
- Search and filter tasks
- Task statistics
- Mission progress tracking
- Calendar widget
- Dynamic greeting based on time
- Dark / Light mode
- LocalStorage persistence
- Responsive dashboard layout

---

# 🛠 Tech Stack

## Frontend

- React
- JavaScript (ES6+)
- CSS3
- Vite
